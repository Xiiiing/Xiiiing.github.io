    数据集列表

    数据集名称 下载链接 详细描述
    TCGA GDC Data Portal The Cancer Genome Atlas (TCGA) 是一个由美国国家癌症研究所 (NCI) 和国家人类基因组研究所 (NHGRI) 联合发起的癌症基因组学项目，涵盖超过 11,000 个初诊癌症和匹配的正常样本，涉及 33 种癌症类型。数据集包括基因表达、突变数据和临床信息，旨在加速癌症分子机制的理解，支持诊断、治疗和预防研究。数据可通过 NCI Genomic Data Commons 访问，部分数据需要权限申请。
    CoCaHis CoCaHis Website CoCaHis 包含 82 张 H&E 染色显微图像，来自 19 名患者的转移性结肠癌肝脏标本，采集于 2017 年 3 月至 2020 年 2 月。图像由克罗地亚萨格勒布 Dubrava 临床医院病理学部提供，包含像素级标注（Fleiss' kappa = 0.74），由四名病理学家、两名病理学住院医师和一名医学生完成。数据集支持手术中计算机辅助诊断 (CAD)，特别是针对转移性结肠癌的快速诊断。
    BCNB Grand Challenge Early Breast Cancer Core-Needle Biopsy WSI Dataset (BCNB) 包含 1058 名早期乳腺癌患者的核心针活检全切片图像 (WSI)，由两名独立病理学家标注。数据集提供临床特征，如年龄、肿瘤大小、ER/PR/HER2 状态等，分为训练集 (60%)、验证集 (20%) 和测试集 (20%)，用于开发乳腺癌淋巴结转移预测算法。访问可能需要注册。
    CAMELYON17 Grand Challenge CAMELYON17 数据集包含 1000 张 H&E 染色淋巴结全切片图像，来自荷兰五个医疗中心，旨在检测乳腺癌转移。数据集包括 100 个训练样本和 100 个测试样本，每位患者提供 5 张切片，标注由专家病理学家监督，采用免疫组化验证。数据以 TIFF 格式提供，支持使用 OpenSlide 或 ASAP 工具处理，适用于开发自动化转移检测算法。
    MIDOG2022 Grand Challenge MIDOG2022 是 Mitosis Domain Generalization Challenge 2022 的数据集，包含 405 个病例，涉及 6 种肿瘤类型（乳腺癌、犬淋巴瘤、神经内分泌肿瘤等），共 9501 个有丝分裂图标注。图像为 2mm² 区域的 TIFF 文件，旨在开发跨肿瘤类型、实验室和物种的有丝分裂检测算法。数据集扩展为 MIDOG++，包含 503 个病例和 12,000 个标注。
    AGGC2022 AGGC2022 Access Form AGGC2022 数据集包含前列腺癌的 H&E 染色全切片图像，专注于识别不同 Gleason 模式，研究不同数字扫描仪的图像变化影响。数据集是数字病理学中首个探讨扫描仪变异性的挑战，数据需通过申请表格获取，下载链接将在 15 个工作日内发送。
    ARCH University of Warwick ARCH 数据集由华威大学提供，包含组织切片图像，专注于数字病理学中的图像分析和模式识别。详细信息可访问其官方网站获取。
    BACH Zenodo BACH (Breast Cancer Histology) 数据集包含乳腺癌组织切片图像，旨在支持乳腺癌诊断和分级研究。数据以高分辨率图像形式提供，详细信息可在其 Zenodo 页面找到。
    CAMEL Google Drive CAMEL 数据集专注于乳腺癌组织切片图像的检测和分类，包含高分辨率图像，适用于开发自动化诊断算法。详细信息可通过 Google Drive 链接访问。
    LC2500 Academic Torrents LC2500 数据集包含肺癌和结肠癌组织切片图像，旨在支持癌症图像分析和分类研究。详细信息可在其 Academic Torrents 页面找到。
    MIDOG2021 IMIG Website MIDOG2021 是 Mitosis Domain Generalization Challenge 2021 的数据集，包含乳腺癌组织切片图像，专注于跨扫描仪的有丝分裂检测。数据包括多个肿瘤类型的标注，详细信息可在其官方网站找到。
    OCELOOT Zenodo OCELOOT 数据集包含组织切片图像，专注于癌症检测和分类，数据以高分辨率格式提供。详细信息可在其 Zenodo 页面找到。
    Osteo The Cancer Imaging Archive Osteo 数据集包含骨肉瘤患者的成像数据，旨在支持骨肉瘤诊断和治疗研究。数据由 Cancer Imaging Archive 提供，详细信息可在其页面找到。
    PAIP2019 Grand Challenge PAIP2019 是 Pathology AI Platform (PAIP) 的挑战赛数据集，包含肝癌全切片图像，专注于肝癌区域分割和活性肿瘤负荷估计。数据需注册获取，详细信息可在其官方网站找到。
    PAIP2020 Grand Challenge PAIP2020 包含结肠癌全切片图像，专注于检测肿瘤区域并分类微卫星不稳定性 (MSI) 分子亚型。数据集包括 47 个训练样本、31 个验证样本和 40 个测试样本，需注册下载。
    PAIP2021 Grand Challenge PAIP2021 包含 240 张结肠、前列腺和胰腺的全切片图像，由专家病理学家标注，专注于多器官癌症的周围神经侵袭检测。数据需注册获取，详细信息可在其官方网站找到。
    SICAHR2 Mendeley Data SICAHR2 数据集包含皮肤癌组织切片图像，专注于皮肤癌检测和分类。详细信息可在其 Mendeley 数据页面找到。
    SCARPOK Zenodo SCARPOK 数据集包含皮肤癌组织切片图像，专注于皮肤癌检测和分类。详细信息可在其 Zenodo 页面找到。
    PCam GitHub PCam 数据集包含乳腺癌组织切片图像，专注于乳腺癌检测和分类，适用于开发深度学习模型。详细信息可在其 GitHub 页面找到。
    HistoGen GitHub HistoGen 数据集包含大量组织切片图像，专注于组织生成和图像合成研究。详细信息可在其 GitHub 页面找到。
    PathGen-Instruct Hugging Face PathGen-Instruct 是一个基于指令的组织生成数据集，专注于通过指令生成组织切片图像。详细信息可在其 Hugging Face 页面找到。
    PathCap Hugging Face PathCap 数据集包含组织切片图像，专注于组织切片的分类和检测。详细信息可在其 Hugging Face 页面找到。
    OPENPATH Google Drive OPENPATH 是一个开放的组织切片图像数据集，专注于支持数字病理学研究。详细信息可通过 Google Drive 链接访问。
    Quilt-1M GitHub Quilt-1M 是最大的视觉语言组织病理学数据集，包含 100 万对图像-文本样本，从 1087 小时的 YouTube 教育视频和其他来源提取，涵盖 438,000 张独特图像和 768,826 个文本对，适用于多模态学习研究。
    PathMMU Hugging Face PathMMU 是一个多模态组织病理学数据集，专注于支持多模态学习和分析。详细信息可在其 Hugging Face 页面找到。
    OCEOT Grand Challenge OCEOT 数据集包含组织切片图像，专注于癌症检测和分类，数据来自 2023 年 OCEOT 挑战赛。详细信息可在其官方网站找到。
    Gleason_CNN GitHub Gleason_CNN 数据集包含前列腺癌组织切片图像，专注于 Gleason 分级的自动化研究。详细信息可在其 GitHub 页面找到。
    VALSET Zenodo VALSET 是一个验证数据集，用于评估组织病理学模型的性能。详细信息可在其 Zenodo 页面找到。
    PAIP23 Grand Challenge PAIP23 是 Pathology AI Platform (PAIP) 的最新挑战赛数据集，专注于数字病理学中的前沿研究。详细信息可在其官方网站找到。
    NASnetLarge Zenodo NASnetLarge 数据集基于神经架构搜索，包含组织病理学图像，专注于支持自动化模型设计。详细信息可在其 Zenodo 页面找到。
    RCDpia (KIRC, KICH, KIRP) RCDpia Website RCDpia 数据集包含肾癌（KIRC、KICH、KIRP）组织切片图像，专注于肾癌检测和分类。详细信息可在其官方网站找到。
    CATCH The Cancer Imaging Archive CATCH 数据集包含癌症成像数据，专注于支持癌症成像研究。详细信息可在其 Cancer Imaging Archive 页面找到。
    SkinCancer heiDATA SkinCancer 数据集包含皮肤癌组织切片图像，专注于皮肤癌检测和分类。详细信息可在其 Heidelberg 数据页面找到。
    MHIST BMIRDS GitHub MHIST 是一个多模态组织病理学数据集，专注于支持多模态学习和分析。详细信息可在其官方网站找到。
    WSSSLUAD Grand Challenge WSSSLUAD 是一个弱监督的肺腺癌组织切片图像数据集，专注于支持弱监督学习研究。详细信息可在其官方网站找到。
    LC25000 GitHub LC25000 数据集包含肺癌和结肠癌组织切片图像，专注于支持癌症图像分析研究。详细信息可在其 GitHub 页面找到。
